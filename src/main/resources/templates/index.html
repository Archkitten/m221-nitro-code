TaskE

What is TaskE?

TaskE was designed as a to-do list project, built to keep track of assignments with the help of your friends.
Our planned features are to have:
  1. A completed Home page and About Us page.
  2. A way to sign up or log in, as well as a way to set friends.
  3. A personal task board, with a way to create new tasks, check tasks as complete, and switch the view from weekly to monthly.
  4. The different types of tasks can be weekday tasks, due date tasks, everyday tasks, weekly tasks, monthly tasks, weekend tasks, and linked tasks.
  5. There might also be a way to follow a group task board, and recieve tasks from it.
  6. There is also planned to be a leaderboard that can help with task completion and motivation.

Why was TaskE created?

TaskE was created as a project for AP Computer Science A, helping build the collaboration for our team and supporting our intended audience.

Will TaskE be finished or scrapped?

It is possible that TaskE might need to be scrapped in the future if it doesn't fit or if we're unable to do it.
However, we're working hard to make this website idea a reality!
